ProgSnap 2

ProgSnap 2 is a proposed specification for datasets containing student programming snapshot data. The specification is not complete, but is approaching a point where data can be exported and analyses can be performed.

ProgSnap 2 was created by the members of the SPLICE Small Code Snapshots working group. The group leaders are Kelly Rivers and David Hovemeyer.
